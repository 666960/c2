Detailed Description

Defines a checkbox UI control. This is a button that toggles between two icons when clicked.

The following properties are available for checkboxes:

checkBox <CheckBox id>=""> { style = <Style id>=""> position = <x, y> size = <width, height> text = <string> checked = <bool> iconSize = <width, height> // The size to draw the checkbox icon, if different from its size in the texture. }
