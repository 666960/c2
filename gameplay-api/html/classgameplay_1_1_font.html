Font * gameplay::Font::create ( const char *  path,
const char *  id = NULL 
) [static]

Creates a font from the given package.

If the 'id' parameter is NULL, it is assumed that the Package at 'path' contains exactly one Font resource. If the Package does not meet this criteria, NULL is returned.

If a font for the given path has already been loaded, the existing font will be returned with its reference count increased.

Parameters:
pathThe path to a package file containing a font resource.
idAn optional ID of the font resource within the package (NULL for the first/only resource).
Returns:
The specified font.
