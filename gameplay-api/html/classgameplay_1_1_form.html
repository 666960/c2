void gameplay::Form::setNode ( Node *  node)

Attach this form to a node.

A form can be drawn as part of the 3-dimensional world if it is attached to a node. The form's contents will be rendered into a framebuffer which will be used to texture a quad. This quad will be given the same dimensions as the form and must be transformed appropriately. Alternatively, a quad can be set explicitly on a form with the setQuad() methods.

Parameters:
nodeThe node to attach this form to.
