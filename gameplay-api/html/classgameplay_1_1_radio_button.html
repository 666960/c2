Detailed Description

Similar to a checkbox, a radio button can be toggled between two states.

However, a radio button can belong to a group, and only one radio button from a group can be selected at one time.

The following properties are available for radio buttons:

radioButton <RadioButton id>=""> { style = <Style id>=""> position = <x, y> size = <width, height> text = <string> group = <string> iconSize = <width, height> // The size to draw the radio button icon, if different from its size in the texture. }
